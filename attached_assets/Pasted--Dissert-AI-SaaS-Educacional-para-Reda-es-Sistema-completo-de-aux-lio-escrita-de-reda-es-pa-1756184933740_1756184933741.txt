# Dissert AI - SaaS Educacional para Redações
**Sistema completo de auxílio à escrita de redações para concursos e vestibulares**

## 1. VISÃO GERAL DO PRODUTO
O **Dissert AI** é uma plataforma SaaS inovadora que revoluciona o processo de aprendizagem e escrita de redações para concursos e vestibulares. Utilizando inteligência artificial avançada, o sistema oferece um ecossistema completo que acompanha o estudante desde a pré-escrita até a finalização do texto, com funcionalidades pedagógicas baseadas em teorias de andaime cognitivo e aprendizagem personalizada.

### Funcionalidades Centrais:
- **Arquiteto de Argumentos:** Diálogo socrático para construção de argumentos sólidos através de perguntas guiadas que desconstróem o tema proposto
- **Explorador de Repertório:** Banco de dados inteligente de referências atualizadas conectado a fontes de notícias, artigos acadêmicos e bases de dados estatísticas
- **Criador de Estrutura Personalizada:** Análise do estilo de escrita do usuário para gerar estruturas de redação modelo que incorporam suas preferências
- **Newsletter Educacional:** Curadoria semanal de temas relevantes com análise contextualizada para provas
- **Controlador de Estilo:** Ajustes interativos de formalidade, tom e complexidade vocabular com visualização em tempo real
- **Simulador de Prova:** Ambiente realista que reproduz as condições reais de exames com cronômetro e feedback imediato
- **Criador de Propostas:** Ferramenta para elaborar temas de redação com textos motivadores
- **Biblioteca Pessoal:** Repositório inteligente de todo o aprendizado do usuário

## 2. DIRETRIZES DE DESIGN E USABILIDADE
### 2.1 Design System
- **Cores Primárias:**
  - Azul escuro: `#09072e` (fundos, títulos, elementos de destaque)
  - Azul vivo: `#5087ff` (CTAs, ícones ativos, botões primários)
  - Cinza suave: `#adacb7` (textos secundários, bordas sutis)
  - Branco puro: `#ffffff` (fundos de cards, texto sobre cores escuras)
  - Gradientes: Combinações suaves entre azul escuro e azul vivo
- **Tipografia:** Inter (Google Fonts)
  - **Títulos principais:** Bold, 32-40px
  - **Subtítulos:** Medium, 24-28px
  - **Corpo de texto:** Regular, 16px
  - **Textos secundários:** Regular, 14px
  - **Labels e botões:** Medium, 14-16px
- **Estilo "Liquid Glass":**
  - Efeito principal: `backdrop-filter: blur(20px)` + `background: rgba(255, 255, 255, 0.85)`
  - Sombras elegantes: `box-shadow: 0 20px 40px rgba(9, 7, 46, 0.15)`
  - Bordas arredondadas: 16px para cards grandes, 12px para elementos menores
  - Transições suaves: `transition: all 0.3s cubic-bezier(0.4, 0, 0.2, 1)`

### 2.2 Componentes Padrão
- **Botões Primários:** Fundo `#5087ff`, texto branco, hover com transform scale(1.02)
- **Botões Secundários:** Borda `#5087ff`, texto `#5087ff`, hover com fundo `rgba(80, 135, 255, 0.1)`
- **Cards:** Fundo branco translúcido, padding 24px, sombra liquid glass
- **Inputs:** Borda `#adacb7`, focus com borda `#5087ff` e glow sutil
- **Navegação:** Sticky header com blur backdrop, transições suaves

### 2.3 Princípios de UX
- **Fluxo Natural:** Progressão lógica entre funcionalidades
- **Feedback Imediato:** Respostas visuais instantâneas
- **Personalização Inteligente:** Adaptação baseada no comportamento do usuário
- **Acessibilidade Total:** Compatível com leitores de tela e navegação por teclado
